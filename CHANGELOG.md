# Changelog

All notable changes to this project will be documented in this file.

This project follows semantic versioning. Releases are cut with `npm version` and published as GitHub Releases from tags.

## [0.1.0] - 2026-06-27

### Added

- Initial portable, auth-free Pi setup snapshot.
- Cross-platform Node CLI for `npx` usage.
- macOS/Linux shell scripts for apply, export, and security scan.
- Windows PowerShell scripts for apply, export, and security scan.
- Conservative security scan for working tree and reachable Git history.
- Documentation for security model, local package paths, and Pi extensions in use.
